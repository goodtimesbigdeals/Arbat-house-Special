Гостевые цифровые решения для Arbat House (Vanilla.js демо)

Цель: быстрый рост прямых бронирований, допродаж и репутации без тяжёлых интеграций. Всё ниже — рабочие мини‑модули для мгновенного демо и пилота.

  • Без фреймворков: чистый HTML/CSS/JS.
  • Хранение данных: локально в браузере (LocalStorage).
  • Печать/экспорт: системная печать «в PDF».
  • QR‑генерация: через публичный API qrserver (без ключей).

AI‑консьерж (демо‑чат на правилах)
